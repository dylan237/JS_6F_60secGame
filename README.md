## BOSS弱點

1.【特定技術】遊戲規則  
a.0~20 秒為 1位數計算 (5-3)，21~40 秒為 2 位數計算 (30*19)，41~60 秒為 3 位數計算 (332+312)，加減乘除規則請用隨機產生，不可寫死題目，60 秒內可無限次數答題。  
b.0~40 秒答對加一分，41~60 秒答對加五分，答錯扣一分，最多僅能扣到零分  

2.【特定技術】不可設計跳轉頁面，都得在同一頁內部切換頁面完成。
